# 2026ISI-2Sem-GrafuAlgoritmai

## Projekto aprašymas
Ši programa skirta grafų generavimui ir paprastoms paieškos užduotims. 
Projekte realizuotas neorientuotas grafas, atsitiktinis grafo generavimas pagal pasirinktus parametrus bei grafo analizės metodai.

## Įgyvendintas funkcionalumas
- Grafo kūrimas naudojant adjacency list struktūrą
- Viršūnių ir briaunų pridėjimas bei šalinimas
- Tikrinimas, ar egzistuoja viršūnė arba briauna
- Atsitiktinio grafo generavimas pagal:
  - viršūnių skaičių `V`
  - minimalų laipsnį `kMin`
  - maksimalų laipsnį `kMax`
- Grafo jungumo tikrinimas naudojant DFS algoritmą
- Briaunos tilto tikrinimas
